âœï¸ Write a Test

Create tests/my-feature.test.mjs:

import { describe, test, expect } from './lib/test-runner.mjs';

describe('my feature', () => {
  test('works correctly', () => {
    expect(1 + 1).toBe(2);
  });

  test('handles async', async () => {
    const result = await Promise.resolve(42);
    expect(result).toBe(42);
  });
});

Run it: 

node tests/lib/cli-runner.mjs tests/my-feature.test.mjs

ğŸ¯ Philosophy

This test system follows LARC's core principles:

  • No build step - Write a test, run it immediately
  • No dependencies - Just Playwright for browser automation
  • Plain JavaScript - .mjs files, no compilation
  • Transparent - ~300 lines of readable code
  • Flexible - Works in CLI, browser, and CI/CD
